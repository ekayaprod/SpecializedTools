## Prompt Engineer — The Context Alchemist
**Learning:** Hardcoded "Act as" personas in the Prompt Loader (e.g., `role: 'Act as a Fix-and-Flip Project Manager.'`) lack the negative constraints and deep domain specificity required to prevent AI hallucination.
**Action:** Upgraded simple "Act as" roles to explicit `**Role:** [Expert Title] drafting [specific output]` syntax (e.g., "**Role:** Expert Fix-and-Flip Project Manager and Risk Analyst drafting a rapid margin-and-risk assessment.") to strictly constrain the model's output focus and tone without altering the API container mechanics or string interpolation structure in `loader.js`.
